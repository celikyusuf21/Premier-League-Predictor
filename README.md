# ⚽ Premier League AI Predictor Bot

A Telegram bot that generates football match insights and prediction signals using real match data from API-Football.

The bot can show upcoming Premier League matches and generate AI-based prediction signals.

---

## 🚀 Features

* ⚽ Get upcoming Premier League matches
* 🤖 AI-based match prediction signals
* 📊 Win / Draw / Lose probability analysis
* 💰 Value edge calculation
* 📱 Telegram bot interface

---

## 🧠 Example Signal

```
🚨 AUTO VIP SIGNAL

🏠 Home Win: 13.0%
🤝 Draw: 74.67%
🚀 Away Win: 12.33%

⭐ Confidence: 13.0%
⚡ Signal Strength: LOW
💰 Value Edge: -34.62%
```

## 🤖 Telegram Commands

```
/start
Start the bot

/matches
Show upcoming Premier League matches

/signal
Generate AI prediction signal
```

## 📦 Installation

Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/premier-league-predictor.git
```

Go into the project folder:

```
cd premier-league-predictor
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root.

Example:

```
TELEGRAM_TOKEN=your_telegram_bot_token
FOOTBALL_API_KEY=your_api_football_key
```

---

## ▶️ Run the Bot

Start the bot with:

```
python bot.py
```
